### The Mastro web framework

An minimal (~700 lines of TypeScript) server and static site generator built from first principles.
Zero client-side JavaScript by default and no bundler.
See the [GitHub repo](https://github.com/mastrojs/mastro/) for more info and examples.
To start a new Mastro project, install [Deno](https://deno.com/) and run:

```
deno run --allow-write jsr:@mastrojs/mastro@0.0.5/init
```
