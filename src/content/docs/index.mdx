### The Mastro web framework

A modern server and static site generator built from first principles.
Zero client-side JavaScript by default and no bundler ‚Äì see the [GitHub repo](https://github.com/mastrojs/mastro/).
To start a new Mastro project with [Deno](https://deno.com/), run:

```
deno run --allow-write jsr:@mastrojs/mastro@0.0.5/init
```
