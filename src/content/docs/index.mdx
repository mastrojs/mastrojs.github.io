### The Mastro web framework

An minimal (~700 lines of TypeScript) server-side web framework and static site generator built from first principles.
Zero client-side JavaScript by default and no bundler. Runs on [Deno](https://deno.com/) (Bun, Node.js and CloudFlare Workers are in the works).
See the [GitHub repo](https://github.com/mastrojs/mastro/) for more info and examples.
To start a new Mastro project, paste the following into your terminal:

```
deno run --allow-write jsr:@mastrojs/mastro@0.0.5/init
```
