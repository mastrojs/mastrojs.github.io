Use plain web standards wherever possible. Mastro provides the few missing pieces: a router, a way to generate HTML, and scripts for static site generation and starting a server. The [source code is only ~800 lines](https://github.com/mastrojs/mastro/tree/main/src#readme) of very readable TypeScript.
